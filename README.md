## simulcrum is a DNS simulator that replaces query responses with a configurable address

### Features
- Configurable response address
- Liveness check for upstream DNS (optional)
- Logging for debugging and monitoring
- Simple and lightweight implementation

### Usage
1. Configure the response address in the configuration file.
2. Enable or disable the liveness check for upstream DNS.
3. Start the simulcrum server.
4. Query the DNS server and observe the replaced responses.
